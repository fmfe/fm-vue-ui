## Button 按钮
常用的操作按钮。

### 基础用法

基础的按钮用法。

:::demo 使用`type`、`plain`和`round`属性来定义 Button 的样式。

```html
<div>
  <fm-button>默认按钮</fm-button>
  <fm-button type="primary">主要按钮</fm-button>
  <fm-button type="info">信息按钮</fm-button>
  <fm-button type="success">成功按钮</fm-button>
</div>
```
:::

### 禁用状态

按钮不可用状态。
